# 情绪与音乐规则

## 硬性禁止
- 永远不要播重金属、死亡金属、噪音音乐
- 永远不要播节奏特别拖沓的慢歌（除非是深夜）
- 不要连续播两首同一个艺术家的歌

## 天气联动
- 晴天 → 节奏明快，轻松愉悦
- 阴天 / 多云 → 内敛的爵士或后摇
- 下雨天 → 钢琴曲、雨声 Ambient、日系治愈系
- 冬天寒冷 → 温暖的声学吉他或管弦乐

## 时间联动
- 整点播新歌时，先用一句话简短交代时间和当下状态
- 早晨第一首歌，用一句话说说今天的感觉
- 深夜不要说太多话，音乐自己说

## 日程联动
- 如果今天有重要会议，工作时段给我专注的纯音乐
- 如果是假期，可以随机推荐一些新发现

## 连续播放逻辑
- 每首歌结束后，根据当前曲子的 BPM 和情绪选择相近或渐进的下一首
- 避免情绪断崖式跳变（不要从慢板直接跳到快节奏）
